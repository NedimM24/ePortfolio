# React Ecommerce Application

## Overview

A front-end e-commerce web application built with React. This project demonstrates reusable components, responsive design, client-side routing, state management, and testing practices.

## Links

GitHub Repository:
https://github.com/NedimM24/react-marketplace

Live Demo:
https://react-marketplace-pi.vercel.app/

## Technologies Used

- React
- React Router
- JavaScript
- CSS Modules
- Vitest
- React Testing Library

## Features

- Product browsing
- Shopping cart functionality
- Add, remove, and update cart items
- Quantity management
- Client-side routing
- Responsive design
- Component-based architecture
- Unit testing

## Skills Demonstrated

- Front-End Development
- Component-Based Architecture
- State Management
- UI Design
- Testing and Debugging

## Challenges

Managing shared cart state between components, preventing duplicate products, and keeping quantity updates consistent.

## Future Improvements

- Backend integration
- User authentication
